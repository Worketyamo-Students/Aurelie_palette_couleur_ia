# Palette AI  

**Palette AI** est une application web qui génère automatiquement des palettes de couleurs à partir d’un mot-clé grâce à l’IA.  

## 🚀 Fonctionnalités  

### Gratuit  
- Génération de **3 palettes/jour**.  
- Palette de **4 couleurs** avec codes hex copiables ✅.  
- Bannières de limite + bouton *Passer en Premium*.  

### Premium  
- Générations **illimitées**.  
- **Sauvegarde** des palettes.  
- **Tableau de bord Favoris** (renommer/supprimer).  
- **Personnalisation avancée** (saturation, luminosité).  

### Pages  
- **Accueil** : description + inscription/connexion.  
- **Premium** : présentation des avantages, plans et FAQ.  
- **Générateur** : génération de palettes.  
- **Favoris** : gestion des palettes sauvegardées.  



# Palette AI  

**Palette AI** is a web application that automatically generates color palettes from a keyword using AI.  

## 🚀 Features  

### Free  
- Generate up to **3 palettes/day**.  
- Each palette contains **4 colors** with clickable hex codes ✅.  
- Daily limit banner + *Upgrade to Premium* button.  

### Premium  
- **Unlimited** palette generations.  
- **Save** palettes to user account.  
- **Favorites Dashboard** (rename/delete saved palettes).  
- **Advanced customization** (adjust saturation and brightness).  

### Pages  
- **Home**: project description + signup/login.  
- **Premium**: showcase of benefits, pricing plans, and FAQ.  
- **Generator**: create palettes from keywords.  
- **Favorites**: manage and organize saved palettes.
